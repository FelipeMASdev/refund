# Refund

Aplicação web desenvolvida durante a formação Fullstack da Rocketseat.
A aplicação gerencia solicitações de reembolso de despesas, com categorização por tipo, formatação automática de valores em BRL e atualização dinâmica do total.

## Sobre o projeto

O Refund permite registrar despesas com nome, categoria e valor. Os itens são adicionados dinamicamente a uma lista lateral com ícone correspondente à categoria, e o total é recalculado a cada adição ou remoção.

## Funcionalidades

- Formulário para registrar nome, categoria e valor da despesa
- Formatação automática do campo de valor para o padrão BRL (R$)
- Categorias disponíveis: Alimentação, Hospedagem, Serviços, Transporte e Outros
- Ícone visual correspondente a cada categoria na lista de despesas
- Contador dinâmico de número de despesas registradas
- Cálculo e exibição do total em tempo real
- Remoção individual de despesas com atualização automática do total
- Limpeza automática do formulário após cada adição
